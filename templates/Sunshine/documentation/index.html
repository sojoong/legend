A) HTML structure - top

This template using Bootstrap. All the information within main content is nested within a div with id- name "content". If you need more information you can read Bootstrap documentation on this link

 

1. Sample html structure for content
<div id="content">
	<div class="container">
		<div class="row">
			<div class="span4">
				this is your content			
			</div>
			<div class="span8">
				this is your content
			</div>
		</div>
	</div>
</div>

 

2. You can create column from 12 predefined grid size, then insert your content inside.
<div class="row">
  	<div class="span1">your content</div>
	<div class="span2">your content</div>
	<div class="span3">your content</div>
	<div class="span4">your content</div>
	<div class="span5">your content</div>
	<div class="span6">your content</div>
	<div class="span7">your content</div>
	<div class="span8">your content</div>
	<div class="span9">your content</div>
	<div class="span10">your content</div>
	<div class="span11">your content</div>
	<div class="span12">your content</div>
</div>

B) CSS Files - top

These are css files used in this template:
  1. style.css - main css files (you can edit color, font, etc from this css)
  2. bootstrap.css - css by Twitter
  3. bootstrap-responsive.css - css by Twitter for mobile version
  4. prettyPhoto.css - used for image popup
  5. responsiveslides.css - used for slider
  6. main.css - used to load all css files

You can change almost all styles from style.css file.

E) Insert Images - top

This template using 'lazyload' plugin to load images smoothly when page scrolling. To add new image insert image url in data-original attribute and keep src value as below: 
<img data-original = "my-image-here.jpg" src = "pic-blank-1.gif" alt = "">
